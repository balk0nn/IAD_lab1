# Лаба 1 по инструментам анализа данных
**Задачи по мэтчингу таймстемпов и стрима медианы скользящего окна**

## Сборка если нужны контейнеры
```cmd  
git clone <repository-URL>  
cd IAD_lab1  
docker-compose up --build --remove-orphans
```

## Запуск контейнера
```cmd
docker-compose run <swm/tm>
```

## Что можно делать в контейнерах:
 - Запускать тесты:
    ```cmd
    pytest
    ```
 - Построить графики в консоли:
   ```cmd
   docker-compose run --rm <mt/swm>
   python console_plotting.py
   ```

  - Запустить бенчмарк снова если меняли seed и пересобирали контейнеры
    ```cmd
    python benchmark.py
    ```
  - Запустить main с тестом производительности для задачи mt
    ```cmd
    python main.py
    ```
## Если просто склонировали проект и не собирали контейнеры:
- Можно рисовать уже красивые графики запуская нужные файлы
- Ну и разумеется менять seed и пределы бенчмаркинга без необходимости пересобирать контейнеры
